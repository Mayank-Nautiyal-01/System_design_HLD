# IP Addresses

**Topic**: IP Addresses (Internet Protocol Address)

## Yeh kya hota hai? (What is it?)
Jasise har ghar ka ek unique address hota hai (jaise House No. 12, ABC Society, Delhi), waise hi Internet par har device (Mobiles, Laptops, Servers) ka ek unique address hota hai. Is address ko **IP Address** kehte hain. 

Jab bhi ek device dusre device se internet par communicate karna chahta hai, toh usko IP address ki zaroorat padti hai, taaki data sahi jagah par pahunch sake.

## Types of IP Address
Mainly do types ke IP addresses hote hain:
1. **IPv4 (version 4)**: Purana aur sabse zyada use, ye 32-bit ka hota hai. Dekhne mein kuch aesa lagta hai: `192.168.1.1` ya `104.244.42.1`. Isme numbers ki ek limit hai (~4 billion addresses).
2. **IPv6 (version 6)**: Kyunki duniya mein internet devices badhte ja rahe hain aur IPv4 addresses khatam ho rahe hain, isliye IPv6 laya gaya. Ye 128-bit ka hota hai aur dikhne mein thoda complex hota hai (`2001:0db8:85a3:0000:0000:8a2e:0370:7334`).

Ye "Private" (Local network ke liye) aur "Public" (Internet par identity ke liye) bhi hote hain.

## Example
Aapne Amazon.com se ek courier order kiya. Amazon ko aapke ghar packet (data) bhejne ke liye aapka Home Address (IP Address) chahiye. 

Technical terms mein: Jab aap WhatsApp par message bhejte ho, toh aapka message WhatsApp ke Server ke IP address par jata hai, aur phir server us message ko aapke dost ke phone ke IP address par transfer kar deta hai. 

## Key Points:
* Har device jo net se connect hota hai, uske paas at least ek IP address hota hai.
* Iske bina internet par devices apne data ko sahi jagah send aur receive nahi kar sakte.
